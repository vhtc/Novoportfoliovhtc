Movie Watchlist

  • Category: Flutter, API, Android Studio
  • Client: YellowStack
  • Project date: July 2021 to August 2021
  • Project URL: View on Github

A flutter based application made by Yayady S for YellowStack in its Placement round 1. The application is added with an API service for a widen range with an infinite scrolloption with add/delete/save option in any movie. The App provides the user with an infinite list of movies which are integrated from an open API. The app also gives a description and ratings to enhance the user experience.
